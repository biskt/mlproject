## 개발환경 구축하기
```
mkdir mlmp
```
폴더 생성 후
```
git clone "깃허브url"
```
#### 자신의 깃허브와 연결시켜준다.
#### 그 다음 .gitignore 와 readme.md, pyproject.toml 파일 생성 후 프로젝트를 진행할 titanicproject.ipynb파일을 만든다.
#### 그 다음 주피터노트북으로 넘어가 프로젝트를 진행한다.
